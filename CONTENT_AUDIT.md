## 3. RECOMMENDED SITEMAP (NEW SITE)

### Philosophy
- Flatten the hierarchy — fewer clicks to key conversions
- Merge FAQ into service pages and a global FAQ — eliminate standalone FAQ page that dilutes depth
- Elevate portfolio as a first-class experience
- Add a dedicated Process page to build trust
- Create room for a local SEO "Service Areas" page
- Keep Blog but reposition as "Journal" or "Insights" to elevate brand perception

### Proposed Structure

```
/                           → Homepage
/about                      → About (enhanced with team, story, values, certifications)
/process                    → Our Process (new — 3-4 step deep dive with visuals)
/work                       → Portfolio / Work (renamed from /portfolio)
  /work/[project-slug]      → Individual project cases (richer case study format)
/design                     → Design Services Hub (overview + navigation to services)
  /design/landscapes        → Landscapes
  /design/hardscapes        → Hardscapes
  /design/masonry           → Masonry
  /design/lighting          → Lighting
  /design/irrigation        → Irrigation
/journal                    → Blog / Journal (renamed, repositioned)
  /journal/[post-slug]      → Individual articles
/contact                    → Contact
/consultation               → Schedule Consultation (simplified from /schedule-consultation)
/service-areas              → Service Areas (new — local SEO page for Greenville + surrounding)
/faq                        → FAQ (keep but restructure as accordion hub, not standalone pages)
/legal/privacy-policy       → Privacy Policy
/legal/terms-of-service     → Terms of Service
```

**Removed / Consolidated:**
- `/careers` — no listings; remove or replace with team culture section on About
- Separate FAQ page for each service → merge into service pages as expandable sections
